## Voraussetzungen

- **Node.js ≥ 16** sowie **npm**, idealerweise über **fnm**, **nvm** oder Homebrew installiert  
- **Git** zum Klonen des Repositories  

---

## Projekt klonen

```bash
git clone https://dein-git-server.de/username/code-inspector.git
cd code-inspector/webapp

# mit npm
npm install

# oder mit Yarn
yarn install

# oder mit pnpm
pnpm install

# npm
npm install axios

# Yarn
yarn add axios

# pnpm
pnpm add axios


# npm
npm run dev

# Yarn
yarn dev

# pnpm
pnpm dev
